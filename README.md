AI Speaking Coach

Overview

AI Speaking Coach is an AI-powered application that helps users improve their spoken English.

The application records speech, converts it to text using Whisper, analyzes grammar using Llama 3 through Ollama, provides fluency scores, tracks progress, and generates PDF reports.

Features

- Voice Recording
- Speech-to-Text Conversion
- Grammar Analysis
- Fluency Scoring
- Speaking Level Classification
- Statistics Dashboard
- Progress Chart
- History Tracking
- PDF Report Generation
- Streamlit Web Interface

Technologies Used

- Python
- Streamlit
- OpenAI Whisper
- Ollama
- Llama 3
- SoundDevice
- SciPy
- ReportLab

How to Run

Install dependencies:

pip install streamlit whisper sounddevice scipy ollama reportlab

Run the application:

py -m streamlit run app.py
